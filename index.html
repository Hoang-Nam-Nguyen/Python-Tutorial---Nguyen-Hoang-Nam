Loading and Manipulating CSV Files

Presented by You

Step 1: Import Libraries


import pandas as pd

Step 2: Load a CSV File


# Load CSV into a DataFrame
df = pd.read_csv('data.csv')

# Display the first 5 rows
print(df.head())

Step 3: Data Manipulation

  • Show DataFrame columns
  • Filter rows where 'Sales' > 1000
  • Print filtered data

# Show columns
print(df.columns)

# Filter rows
high_sales = df[df['Sales'] > 1000]

print(high_sales)

Step 4: Save to a New CSV


# Save filtered data to a new CSV
high_sales.to_csv('high_sales.csv', index=False)
