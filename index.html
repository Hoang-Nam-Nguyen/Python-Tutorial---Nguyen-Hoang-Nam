Loading and Exploring CSV Files

Hoang Nam Nguyen

Why CSV Files Matter

  • 80% of business data is stored in Excel/CSV files
  • Typical data: sales, production, customer records
  • First step in business analytics: loading and exploring data

What You Need

  • CSV File: Common format for business data
  • Python Environment:
    • Jupyter Notebook
    • Google Colab (Online, Free)

Import Libraries


import pandas as pd
import os
import matplotlib.pyplot as plt

Pandas: powerful data analysis library in Python

Matplotlib: quick charts and plots

Load a CSV File


# Check working directory
print(os.getcwd())

# Load CSV file
df = pd.read_csv('business_production_data.csv')

Real-world business data: production records, sales

Basic Data Exploration


# Show columns
print(df.columns)

# Summary statistics
print(df.describe())

Check for Missing Data


# Check for missing values
print(df.isnull().sum())

Important step before analysis

Handling Missing Data


# Drop rows with missing values
df_clean = df.dropna()

# Fill missing values with 0
df_filled = df.fillna(0)

Two common methods: Drop or Fill

Data Filtering Example


# Filter products with SalePricePerUnit > 1000
high_price = df[df['SalePricePerUnit'] > 1000]

print(high_price.head())

Focus on high-value products for strategic insights

GroupBy Example


# Group by Product Category and calculate average SalePricePerUnit
avg_price = df.groupby('ProductCategory')['SalePricePerUnit'].mean()

print(avg_price)

Summarize data by groups

Quick Visualization


# Bar chart of average sale price per category
avg_price.plot(kind='bar')
plt.title('Average Sale Price per Category')
plt.ylabel('Sale Price')
plt.show()

First insights with simple plots

Load Specific Columns


# Load only specific columns
df_selected = pd.read_csv('business_production_data.csv', usecols=['ProductName', 'SalePricePerUnit'])

print(df_selected.head())

Load only what you need to save memory

Quick Question

Why might we load only specific columns?

  • Focus on relevant data
  • Reduce memory usage
  • Faster processing

(Backup) Change Working Directory


# Change directory if file not found
os.chdir('path/to/your/directory')

# Verify
print(os.getcwd())

Useful if file is not in current directory
