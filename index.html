Loading and Manipulating CSV Files

Hoang Nam Nguyen

What You Need

  • CSV File: A file with comma-separated values.
    (e.g., sales data, production data, etc.)
  • Python Environment:
    • Jupyter Notebook (Anaconda)
    • Spyder (Anaconda)
    • Google Colab (Online, Free)

Import Libraries


import pandas as pd
import os

Load a CSV File


# Check the current working directory
print(os.getcwd())

# Basic CSV loading
df = pd.read_csv('business_production_data.csv')

# Display first 5 rows
print(df.head(5))

Basic Data Exploration


# Show columns
print(df.columns)

# Summary statistics
print(df.describe())

# Data types and missing values
print(df.info())

Data Filtering Example


# Filter rows with 'SalePricePerUnit' > 1000
high_price = df[df['SalePricePerUnit'] > 1000]

print(high_price.head())

# Save filtered data
high_price.to_csv('high_price.csv', index=False)

Advanced CSV Loading Options


# Load only specific columns
df_selected = pd.read_csv('business_production_data.csv', usecols=['Product', 'SalePricePerUnit'])

# Load only first 100 rows
df_small = pd.read_csv('business_production_data.csv', nrows=100)

# Specify delimiter if not comma
df_tab = pd.read_csv('business_data.tsv', delimiter='\t')

# Parse dates automatically
df_date = pd.read_csv('sales_data.csv', parse_dates=['OrderDate'])

# Handle missing values
df_na = pd.read_csv('business_data.csv', na_values=['?', 'NA', 'Missing'])

Set Data Types with dtype


# Enforce data types when loading
df_types = pd.read_csv(
    'business_production_data.csv',
    dtype={'SalePricePerUnit': float, 'Quantity': int}
)

print(df_types.dtypes)
